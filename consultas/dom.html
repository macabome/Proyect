Modelo DOM

El Modelo de Objetos del Documento, cuyas siglas son DOM,
es una de programación de aplicaciones para documentos
HTML y XML. El modelo DOM define la estructura lógica de los
documentos y el modo en que se accede a ellos y se los manipula.
Con el Modelo de Objetos del Documento , los programadores pueden
construir documentos, navegar por su estructura, y añadir, modificar
o eliminar elementos y contenido.
